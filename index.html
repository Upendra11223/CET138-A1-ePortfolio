Web Fundamentals Portfolio

This portfolio presents my understanding of HTML, CSS, the Bootstrap framework, and JavaScript. Use the Try Demo buttons to open each live example.

Demo Pages

HTML Demo

HTML

A semantic login page built with pure HTML. Demonstrates structural tags (<header>, <main>, <form>, <footer>), semantic form elements, and accessibility attributes like aria-describedby. No CSS or JavaScript — this is raw HTML showing structure only.

Try Demo

CSS Demo

CSS

The same login page styled with modern CSS. Uses Flexbox and Grid for layout, a gradient background, smooth hover transitions, and responsive design. Demonstrates how presentation can be separated from structure to create a visually polished and accessible interface.

Try Demo

Bootstrap Demo

Bootstrap

A responsive page styled using only the Bootstrap CSS framework. Implements the 12-column grid, utility classes for spacing and alignment, and pre-styled components like cards and buttons. Shows how professional layouts can be achieved quickly without writing custom CSS.

Try Demo

JavaScript Demo

JavaScript

A simple calculator built with vanilla JavaScript. Supports addition, subtraction, multiplication, and division. Demonstrates DOM manipulation, event handling, and basic app state management — showing how interactivity is added to otherwise static HTML/CSS pages.
